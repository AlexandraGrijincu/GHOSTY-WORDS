# Ghosty Words

**Ghosty Words** este o versiunea a proiectului creat inițial la hackathonul **Haufe Group (2026)**. Această variantă a fost adaptată pentru a permite rularea rapidă și testarea funcționalităților principale direct în browser.

### https://alexandragrijincu.github.io/GHOSTY-WORDS/

## Despre Joc
Este un joc educațional interactiv pentru învățarea verbelor în limba engleză (**Past Simple**, **Present Continuous**).
